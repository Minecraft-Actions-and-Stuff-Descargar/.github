### Método 2: Instalación manual del APK o .mcpack
1. Descarga el archivo `.mcpack` o `.apk` desde un sitio confiable (como MCPEDL).
2. Mueve el archivo a:
   - Android: `/games/com.mojang/resource_packs`
   - Windows: `%LocalAppData%/Packages/MinecraftUWP.../LocalState/games/com.mojang/resource_packs`
3. Abre Minecraft y activa el recurso en **Ajustes > Recursos globales**.
4. Reinicia Minecraft para aplicar los cambios correctamente.
